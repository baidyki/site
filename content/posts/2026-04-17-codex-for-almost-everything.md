OpenAI [released](https://openai.com/index/codex-for-almost-everything/) a major update to Codex, used by more than 3 million developers weekly, expanding it from a coding assistant into a general-purpose work agent. The headline feature is background computer use: Codex can now operate macOS apps with its own cursor — seeing, clicking, and typing — while running [multiple agents in parallel](https://www.thurrott.com/a-i/openai-a-i/335030/openai-releases-a-major-update-to-codex) without interfering with the user's work. A new in-app browser lets users comment directly on webpages to give the agent precise instructions, currently aimed at frontend and game development with plans to expand to full browser control.

The update also ships [image generation via gpt-image-1.5](https://community.openai.com/t/introducing-the-new-codex-for-almost-everything/1379125) for mockups and prototyping, over 90 new plugins (Atlassian Rovo, CircleCI, GitLab, Microsoft Suite, Neon, Remotion, Render, and others), and deeper developer workflow support: GitHub PR review comments, alpha SSH for remote devboxes, multi-terminal tabs, and rich file previews. Codex can now schedule future work for itself and wake up automatically to continue long-running tasks across days or weeks. A preview of memory lets it retain preferences and corrections from past sessions. Rolling out now to [desktop Codex users](https://openai.com/index/codex-for-almost-everything/) with ChatGPT accounts; computer use is macOS-only for now, with EU/UK access and Enterprise/Edu memory coming later.

Key points:

- **Background computer use** — Codex operates macOS apps autonomously with its own cursor; multiple agents run in parallel without taking over the user's screen
- **In-app browser** — users comment directly on webpages to provide context and instructions to the agent
- **Self-scheduling** — Codex can schedule future work and wake itself up to continue long-running tasks across days or weeks
- **Image generation** — integrated gpt-image-1.5 for mockups, frontend designs, and game assets inside the same workflow
- **Memory (preview)** — retains preferences, corrections, and gathered context from previous sessions to improve future tasks
- **90+ new plugins** — Atlassian Rovo, CircleCI, CodeRabbit, GitLab, Microsoft Suite, Neon, Remotion, Render, and more
